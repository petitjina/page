## 💪 Git 커밋 컨벤션

| 태그이름 | 설명 |
| - | - |
| env | 개발 환경 관련 설정 |
| feat | 새로운 기능 추가 |
| update | 수정, 추가, 파일 삭제 등 |
| fix | 버그를 고친 경우 |
| design | CSS 등 사용자 UI 디자인 변경 |
| style | 코드 포맷 변경, 세미 콜론 누락, 코드 수정이 없는 경우 |
| refactor | 코드 리팩토링 |
| docs | 문서 수정 (MD 파일) |
| test | 테스트 코드를 생성하거나 수정 하는 경우 |
